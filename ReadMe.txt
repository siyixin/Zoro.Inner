1、项目名称

罗罗诺阿. 佐罗，简称Zoro，目标实现将电影服务器（\\10.63.41.50\share）上的电影以格式化文本的形式输出，然后在外网调用豆瓣API获取电影信息，以便观赏

2、应用项目

2.1 NewBridge.Zoro.FileSearch:内网电影文件夹的检索与输出，预计使用System.IO之类的命名空间，实现对文件夹的搜索和格式化输出
2.2 NewBridge.Zoro.Exchange: 交换系统，实现将内网遍历出的电影结果交换到外网的工作

3、注意事项

3.1 映射网络驱动器，例如我把电影服务器映射为Y: 事实证明网络驱动器映射失败，估计是该exe的执行账户与映射网络驱动器的账户不同导致，还是采用网络路径的方式
3.2 通过配置的方式制定遍历后的文本路径，目前采用txt，后期可以采用json或者其他可读性更好的方式
3.3 据说（人教论坛）海关电邮已经禁止了pop3 和 smtp，内外网交互受挫
